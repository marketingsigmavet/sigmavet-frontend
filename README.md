# Insurai React Conversion (Index-8)

This is a direct React/Vite conversion of the `index-8.html` home page from the Envato template.

## Run locally
```bash
npm install
npm run dev
```

## Notes
- Assets are in `public/assets`
- Original HTML structure is preserved inside React for a fast, accurate conversion
- Original template CSS and JS are loaded so the page keeps the same styling and behavior
